# Kronecker Project

This repository contains research code for studying Kronecker sequences and Zaremba's index.

The main goal is to compute and compare values of `rho` for different choices of `N`, `alpha`, and search bounds. The code includes simple brute-force methods, faster NumPy methods, Lyness-style methods, notebooks, plots, and a small static site explaining the results.

## Files

- `math_help.py`: core math and search functions
- `experiment.py`: experiment helpers
- `usage.py`: small example script
- `phase*.ipynb`: research notebooks
- `simple-site/`: basic web page and plots
